The Top Contributor Network of

pystac

pystac is a foundational library in the Python software ecosystem for the SpatioTemporal Asset Catalog (STAC) specification , a common language to describe geospatial information so it can more easily be worked with, indexed, and discovered. This visualization is based on the excellent work by Nadieh Bremer .

NOTES

Only contributors to pystac are shown on this chart. There's many folks that have contributed to the STAC ecosystem more generally, but the visualization code is built only to support contributors to a single central repository.

The code for this visual and to collect the source data lives at gadomski/pystac-top-contributors.
